Swift 实现金额类的动态“蹦字”效果

看到支付宝等金融类 App 都会搞一个一直“蹦字”的显示动画，就想着用 Swift 实现一下。

## Advantage 框架的优势
* 1.文件少，代码简洁
* 2.不依赖任何其他第三方库
* 3.基于Label使用简单
* 4.具备较高自定义性


## Requirements 要求
* iOS 7+
* Xcode 8+


## Usage 使用方法
### 简单使用
```
@IBOutlet weak var numberLabel: AnimationLabel!
//例如要蹦至 111929
numberLabel.countFromCurrent(to: 111929, duration: 1)

```
